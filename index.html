1.2 The following visualizations explore spatial and temporal variations in air quality, highlighting differences in PM10 and PM2.5 distributions and trends.

library(ggplot2)
library(dplyr)

# Plot how pollutant concentrations change over time
plot_aq <- ggplot(airquality_clean, aes(x = sample_date, y = concentration)) +
  geom_line(aes(color = indicator)) +  # Line plot for time series
  facet_wrap(~ indicator, scales = "free_y") +  # Separate plots for each pollutant
  labs(
    title = "Pollutant Concentrations Over Time",
    x = "Date",
    y = "Concentration"
  ) +
  theme_minimal() +  # Clean plot theme
  theme(axis.text.x = element_text(angle = 45, hjust = 1))  # Rotate x-axis labels for better readability

plot_aq
ggsave("aq_temporal_trends.pdf", plot_aq, width = 10, height = 5)

PM10 declined more sharply with greater variation, while PM2.5 showed a steady and consistent decrease. There is a spike in 2020, when we had covid lockdown, and people have been using more resources at home causing spike in pollutants.

summarize_air_quality <- function(data, group_vars) {
  data |>
    group_by(across(all_of(group_vars))) |>
    summarise(mean_conc = mean(concentration, na.rm = TRUE), .groups = "drop")
}
